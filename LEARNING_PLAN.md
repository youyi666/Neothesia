# 钢琴练级手册
> 最终 Boss：花海（周杰伦）
> 武器：Yamaha P-48 + Neothesia
> 每日练习：20 分钟

---

## 序章：键盘未到（当前阶段）

- [ ] 每天桌上练手指独立性 5 分钟（食指→中指→无名指→小指，依次抬起）
- [ ] 用键盘在 Neothesia 里玩一遍，熟悉界面操作
- [ ] 记住一件事：两个黑键左边的白键是 C

---

## Lv.1 — 新手村（第 1–2 周）

**目标：** 右手能弹完小星星

- [ ] 认识 C D E F G A B 七个白键的位置
- [ ] 练习 C 大调音阶：右手上下各 5 遍，极慢
- [ ] 跟弹 `01_single_hand/01_twinkle_twinkle_right_hand_slow.mid`

**过关条件：** 右手不停顿地跟完小星星 ✓

---

## Lv.2 — 初入江湖（第 3–4 周）

**目标：** 右手流畅，认识更多曲子

- [ ] `01_single_hand/02_ode_to_joy_right_hand_slow.mid`（欢乐颂）
- [ ] `01_single_hand/03_mary_had_a_little_lamb_right_hand_slow.mid`
- [ ] `01_single_hand/04_frere_jacques_right_hand_slow.mid`
- [ ] `01_single_hand/05_au_clair_de_la_lune_right_hand_slow.mid`
- [ ] 左手单独练 C 大调音阶

**过关条件：** 5 首单手曲全部跟完，中间不暂停 ✓

---

## 🏆 Boss 1：单手之王

> 右手已经听话了。接下来训练左手，然后让两只手认识彼此。

---

## Lv.3 — 左手觉醒（第 5–6 周）

**目标：** 左手开始工作

- [ ] 左手单独跟弹 `02_two_hands_easy/01_twinkle_twinkle_two_hands_easy.mid`（只看左手轨道）
- [ ] 双手音阶：右手上行时左手也上行，极慢
- [ ] 每天先右手练一遍，再左手练一遍，最后合并试一次

**过关条件：** 左右手能各自单独跟完小星星双手版 ✓

---

## Lv.4 — 双手协议（第 7–10 周）

**目标：** 双手开始配合

- [ ] `02_two_hands_easy/01_twinkle_twinkle_two_hands_easy.mid` 双手合并
- [ ] `02_two_hands_easy/02_ode_to_joy_two_hands_easy.mid`
- [ ] `02_two_hands_easy/03_amazing_grace_two_hands_easy.mid`
- [ ] Neothesia 播放速度调到 70%，跟不上就继续降

**技巧：** 右手旋律背熟之后，眼睛才能分心看左手。先背，再合。

**过关条件：** 三首双手版能跟完，不断 ✓

---

## 🏆 Boss 2：双手同盟

> 两只手开始说话了。现在学真正好听的东西。

---

## Lv.5 — 优美曲目（第 11–16 周）

**目标：** 弹出有音乐感的旋律

- [ ] `03_beautiful_slow/01_greensleeves_melody_slow.mid`（绿袖子）
- [ ] `03_beautiful_slow/03_scarborough_fair_melody_slow.mid`（斯卡布罗集市）
- [ ] `03_beautiful_slow/02_satie_gymnopedie_theme_easy.mid`（忧愁）
- [ ] `02_two_hands_easy/04_pachelbel_canon_easy_loop.mid`（卡农简化版）
- [ ] 尝试不看 Neothesia 提示，凭记忆弹出绿袖子

**过关条件：** 绿袖子能脱离 Neothesia 提示完整弹出 ✓

---

## Lv.6 — 进阶挑战（第 17–20 周）

**目标：** 接触真正有难度的曲子

- [ ] `04_challenge/01_bach_minuet_g_opening_easy.mid`（小步舞曲）
- [ ] `04_challenge/02_canon_in_d_slow_two_hands.mid`（卡农双手慢速）
- [ ] 开始学习分解和弦伴奏型：C-G-Am-F 和弦，左手逐个音弹

**过关条件：** 卡农双手版能跟完 ✓

---

## 🏆 Boss 3：古典猎人

> 古典小曲已经拿下。现在转向最终目标。

---

## Lv.7 — 花海前传（第 21–22 周）

**目标：** 准备好弹花海需要的技能

- [ ] 左手分解和弦练习：把 C 和弦拆成 C-E-G-E 反复弹
- [ ] 在 Musescore 找花海简化版 MIDI，导入 Neothesia 听一遍
- [ ] 识别花海旋律里的音，右手单独跟弹副歌部分

---

## Lv.8 — 花海（第 23–26 周）

**目标：** 弹出花海

- [ ] 右手旋律：主歌跟完
- [ ] 右手旋律：副歌跟完
- [ ] 左手伴奏型单独练熟
- [ ] 主歌双手合并
- [ ] 副歌双手合并
- [ ] 完整连弹一遍

---

## 🏆 最终 Boss：花海通关

> 六个月前你还不知道 C 在哪里。

## 每天怎么用这个清单

1. 打开这个文件
2. 找到当前所在的等级
3. 做当天的练习
4. 做完在对应 `[ ]` 里打 `x` 变成 `[x]`
5. 关掉文件，去做别的事

**不要跳级。每一关都是下一关的地基。**
