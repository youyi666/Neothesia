## 每天怎么练

每天 15-20 分钟就够：

1. 打开第一首，只用右手练 5 分钟。
2. 播放速度如果仍然快，进播放界面按方向键下调慢。
3. 能跟到 70% 左右再换下一首。
4. 两手版不要急，先把右手旋律练顺。

Neothesia 播放界面常用按键：

- 空格：暂停/继续
- 方向键下/上：降低/提高播放速度
- 方向键左/右：倒退/快进
- PageUp/PageDown：调整瀑布流动画速度

## 来源说明

原始下载主要来自 Mutopia Project。Mutopia 上的音乐通常是 Public Domain 或 Creative Commons 授权；具体授权以每个曲目页面为准。

已下载的原始来源包括：

- Satie - Gymnopedie No. 1
- Mozart - Ah Vous Dirai-Je Maman / Twinkle theme
- Schumann - Album pour la jeunesse, Op. 68 No. 1 Melodie
- Beethoven - Ode to Joy
- Amazing Grace
- Greensleeves

`01_*` 到 `04_*` 目录里的练习版是为了学习跟弹重新整理的简化 MIDI。生成脚本在 `tools/generate_beginner_midis.js`。
